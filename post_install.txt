Here's some things you need to do manually:

    In the Mac App Store:
        - Sign in before running `./install.sh` if you want the Brewfile-managed App Store apps installed automatically

    In iterm2:
        - Update the font to `Source Code Pro for Powerline`
        - Import the settings stored in this project
    
    Install these applications:
        - [Cactus VPN](https://www.cactusvpn.com/)
        - [Logitech Capture](https://www.logitech.com/en-us/software/capture.html)
        - Adobe Suite
        - Wacom
